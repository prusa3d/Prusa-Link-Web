# Prusa-Connect-Local

This is a printer connect webservice. It is compatible with other 3D printer services, so many applications can use this service.

### Install

```bash
npm install
```

## UI

### Run ui sla

```bash
npm run ui:start
```

### Run ui fdm

```bash
npm run ui:start-mini
```

### Build dev

```bash
npm run ui:dev
```

### Build ui

```bash
rm -fr ./dist
npm run ui:build
```

### update translations ui

```bash
npm run ui:translations
```
